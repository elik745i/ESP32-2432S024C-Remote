## v0.2.8

- Added press-and-hold menu reordering with persisted item order across menus and submenus.
- Added global flat/3D button styling, clearer stateful toggle feedback, and the new `AP Mode` home action.
- Moved LVGL allocation on the S3 build to a PSRAM-first allocator to reduce internal SRAM pressure on heavy menus.
