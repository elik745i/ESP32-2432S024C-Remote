## v0.1.4

- Added persistent OTA update checking with a top-bar update indicator across all supported boards.
- Added `Config -> OTA Updates` with current/latest version display, on-device update trigger, and live progress bar.
- Added update-available and post-update confirmation popups, both persisted across reboot as needed.
- Added board-correct OTA asset selection for multi-board releases, including the `ESP32-S3-3248S035-N16R8`.
- Added the `esp32-eyes` project to the README source list.
