## v0.2.4

- Added HC-12 radio chat discovery so devices on the same HC-12 channel appear in chat and peer lists.
- Added HC-12 radio chat/control transport with message queue, ACK, and delete support alongside Wi-Fi and MQTT.
- Encrypted HC-12 chat/control payloads with the same Curve25519/XChaCha20-Poly1305 model already used for Wi-Fi P2P and MQTT.
- Added the ESP32-S3 HC-12 pad wiring photo to the repository and README.
