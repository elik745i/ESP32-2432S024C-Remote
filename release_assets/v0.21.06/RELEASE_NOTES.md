# v0.21.06

## Highlights

- Renamed `Config -> Style` to `Config -> Screen`
- Added `Screen Lock` with a 4-digit PIN setup popup and persisted boot-time lock screen
- Added escalating PIN retry delays after repeated failed unlock attempts plus a lock-screen `Factory Reset` shortcut after 3 failures
- Updated the README to document the current `Screen`, `Radio Control`, and `Remote Control` feature set
