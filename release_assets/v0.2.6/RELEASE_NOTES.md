## v0.2.6

- Added a persisted `Web Server` toggle in `Config > WiFi Config`.
- Turning the web server off now shuts down HTTP, websocket, mDNS, and captive DNS runtime work.
- Rebalanced the main loop so UI touch and animations take priority over background services during interaction.
- Included the latest `New_Case_035_v1.max` 3D model update.
