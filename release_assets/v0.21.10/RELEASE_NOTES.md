# Release Notes

## v0.21.10

- Removed the trailing `ON/OFF` text from MQTT control button labels so the controls screen shows only the configured button names.
- Fixed `Config -> Screen -> Button Style -> Black Buttons` so MQTT toggle controls stay green while the state is ON instead of only flashing green during a press.
- Includes the recent MQTT button-config keyboard visibility improvement when adding or renaming controls.
- Updated firmware version metadata, module manifest versions, and release documentation for the new device firmware release.
