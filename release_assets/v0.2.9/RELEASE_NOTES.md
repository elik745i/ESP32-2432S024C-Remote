## v0.2.9

- Fixed swipe-back reliability on dense menu screens with many buttons.
- Vertical scrollable lists no longer steal back-swipe unless they actually need horizontal gesture ownership.
- Reorderable buttons now claim the gesture only after long-press drag mode starts.
